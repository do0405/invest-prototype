가장 큰 변화: 두 개가 하나로
기존에는 메인 지표 + 보조 지표, 슬롯 2개를 써야 했습니다.
이제 하나의 지표에 다 들어갑니다.
메인 시그널, 보조 시그널(수급/VCP/W패턴/시그마), 성장 지표(EPS/매출), 대시보드까지. 슬롯 1개로 전부 해결됩니다.
보조 시그널 대시보드도 메인 대시보드에 통합했습니다. 이제 하나의 대시보드에서 다 보고, 성장률(EPS/Sales)만 별도 테이블로 보조합니다. 두 개의 대시보드를 왔다 갔다 할 필요가 없어졌고, 신호에 대한 혼란이 최소화됩니다.
대시보드, 왜 상세 모드를 없앴을까?
기존에는 "간소화(MOST)" 모드와 "상세" 모드 두 가지가 있었습니다.
솔직히 상세 모드는 정보가 너무 많았어요. 매수 조건, 종목 상세, 섹터 정보... 데이터는 많은데 "그래서 사야 해, 말아야 해?"에 대한 답이 바로 안 나왔습니다.
상세 모드를 없애는 대신, 간소화 모드를 훨씬 더 알차게 만들었습니다. 기존 상세 모드에서 진짜 필요한 정보만 골라서 간소화 모드에 녹여넣었어요. 그 결과 보조 지표까지 통합할 수 있는 공간이 생겼고, 하나의 지표 + 하나의 대시보드가 가능해진 겁니다.
전설들의 원칙
이번 업데이트의 설계 기반이 된 트레이딩 원칙들입니다.
마크 미너비니 — 추세가 살아있을 때만 매수. Stage 분석. 
윌리엄 오닐 — 하나라도 안 맞으면 사지 않는다. CANSLIM. 
제시 리버모어 — 대세를 거스르지 않는다. 200일선. 
쿨라메기 — T-라인이 꺾이면 매수 금지.
각자 스타일은 다르지만, 핵심에서는 놀랍도록 일치했습니다:
"확실한 자리에서만, 확신을 가지고, 리스크를 관리하며."
새로운 기능들
🎯 확신 등급 (Conviction Grade)
모든 종목에 S / A / B / C / D 등급이 표시됩니다.
단순한 점수 합산이 아닙니다. 먼저 "이 종목을 봐도 되는가?"를 확인합니다. 이평선 배열, 시장 상황, 추세 방향 — 이 중 하나라도 안 맞으면 D등급이 뜨고, 왜 D인지 이유가 함께 표시됩니다. "역배열", "시장 약세" 같은 식으로요.
필수 조건을 통과한 종목에 한해서만 S~C 등급이 매겨집니다. RS(상대강도), 거래량, 52주 고점 거리, 보조 신호 등을 종합합니다.
더 이상 "시그널 떴으니까 일단 해볼까?"가 아니라, 대시보드의 등급을 보고 판단할 수 있습니다.
(본 지표는 정보 제공 목적이며, 특정 종목의 매수/매도를 권유하지 않습니다.)
🏉 풀백 시그널 재설계
기존에는 급등 후 내려오면 다 풀백이었습니다. 하락 구간에서도요.
이제는 여러 조건을 동시에 충족해야 풀백이 뜹니다:
→ 먼저 급등(부분익절/과열)이 있었어야 합니다 → 이평선 근처까지 내려와야 합니다 → 투매가 아닌 건조한 조정이어야 합니다 → 이평선이 여전히 올라가고 있어야 합니다 (꺾이면 하락이지 풀백이 아닙니다) → 마지막으로, 실제 반등이 시작된 양봉에서만 뜹니다
덜 뜹니다. 하지만 뜨면, 이전보다 훨씬 신뢰할 수 있습니다.

🛡️ 이평선 꺾임 차단
이평선(EMA)이 하락 전환하면 진입 시그널이 뜨지 않습니다. 돌파 진입도 마찬가지입니다.
"차트는 좋아 보이는데 이평선이 꺾여있어서 들어갔다가 와리가리 당하는" 상황을 원천적으로 방지합니다.
🔄 EMA 정렬 — 이건 뭔가요?
처음 보시는 분들을 위해 설명드릴게요.
주식에는 이동평균선(이평선)이라는 게 있어요. 쉽게 말하면 "최근 며칠간의 평균 가격"을 선으로 그은 겁니다. 단기, 중기, 장기 이렇게 3개를 씁니다.
정배열 🟢 — 단기 > 중기 > 장기 순서로 위에서 아래로 깔끔하게 정렬된 상태. 건강한 상승 추세입니다.
역배열 🔴 — 반대로 장기가 위에 있는 상태. 하락 추세입니다. 이때는 매수하면 안 됩니다.
꼬임 🟠 — 이평선들이 엉켜있는 상태. 방향을 정하는 중입니다. 좀 더 지켜봐야 합니다.
이번 대시보드에서는 이전 상태에서 현재 상태로의 전환을 보여줍니다:
🟢 정배열 (← 꼬임) — 방금 상승 전환! 관심 구간 
🟠 꼬임 (← 정배열) — 추세가 약해지고 있어요. 주의 
🔴 역배열 (유지) — 계속 하락 중. 관망
이걸 보면 "추세가 어디로 가고 있는지"를 바로 알 수 있습니다.

📊 대시보드 — 위에서 아래로 읽으면 판단 끝
대시보드를 완전히 재구성했습니다. 3가지 질문에 답하는 구조입니다:
[지금 뭐 해야 해?] 시그널 → 확신 등급 → AI 평가
[왜?] EMA 정렬 → 보조 신호 + 기술 점수 → 시장/섹터 → 추세 에너지 → 시장 주도권 → 상대 강도(RS) → 거래량 → 52주 고점%
[리스크는?] 손절 관리 → 목표 수익 → 실시간 손익비
[기업은 건강한가?] 매수 자격 → 펀더멘털 평가

위에서 아래로 읽기만 하면 됩니다.
🚦 보조 신호 — 한눈에 보는 종합
기술 점수와 활성 신호를 한 줄로 통합했습니다.
예시: 🟢 72점 | 수급 VCP PB
신호등 색상으로 즉시 파악하고, 점수로 정량화하고, 어떤 보조 신호가 활성인지까지. 기존에 별도 대시보드에서 확인하던 정보가 메인에 바로 보입니다.
🚫 200일선 매매 금지
설정에서 체크하면 주가가 200일 이동평균선 아래일 때 모든 매수 시그널을 차단합니다. 하락장에서 물리는 걸 원천적으로 방지합니다.
🧸 학습 모드 (멤버지표 전용)
이건 원래 있는 기능입니다만, 처음 사용자 분들이 찾기 쉽게, 설정 맨 위에 체크박스를 뒀습니다.
켜면 모든 매매 시그널에 초보자용 코칭 문구가 추가됩니다.
예를 들어 정석 진입 시그널이 뜨면:
"지금 흐름이 좋아요! 👍 하지만 주식에 100%는 없어요. 손절가부터 정하세요."
부분 익절 시그널이 뜨면:
"수익 일부를 주머니에 넣으세요. 😎 그래야 마음이 편해서 남은 물량으로 더 큰 수익을 노릴 수 있어요!"
처음 접하시는 분들도 시그널의 의미를 이해하면서 자연스럽게 매매 원칙을 익힐 수 있습니다.
두 가지 버전
📚 멤버전용 (멤버전용-Lazy Alpha Indicator)
처음 접하시는 분, 주린이, 매매 원칙을 체계적으로 배우고 싶은 분을 위한 버전입니다.
→ 학습 모드 내장 (체크박스 하나로 코칭 문구 ON) → 설정 항목에 툴팁 (마우스 올리면 설명) → 알림은 TradingView 앱 전용 (심플) → 불필요한 설정 최소화
멤버전용으로 학습용으로 시작하시고, 익숙해지면 자동매매용으로 넘어가시면 됩니다.
🤖 자동매매용 (자동매매용-Lazy Alpha Indicator)
자동매매 봇과 연동하는 전문가용 버전입니다. 자동매매만을 위한게 아니라 학습모드가 이제 필요 없으시고 데이터를 연동해서 사용하실 수 있는 분들은 모두 사용가능한 버전입니다.
→ JSON 25필드 웹훅 (확신 등급 포함) → 실시간 진입 모드 + 진입 무효 감지 → 봇 전용 데이터 (캔들 타입, EMA 터치, 정렬 상태 등, 각종 JSON 제공) 
*학습 모드 없음
