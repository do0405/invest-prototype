# 포지션 관리 및 포트폴리오 관리

### 1. 개별 포지션을 전체 계좌로 연계해서 생각하기

개별 포지션 trade를 전체 계좌로 연계해서 생각해야 합니다. 예를 들어 손익비가 1:3인 주식에 전체 포트 대비 1%를 투자했다면, 수익 시 1%를 벌고 손실 시 0.33%를 잃는 것입니다.

**포지션을 잡을 때, 해당 포지션에 한해 손익비 계산도 하지만, 전체 계좌 대비 손익비를 계산할 때가 더 많습니다.**

### 2. 포지션 사이징 방법

**Pilot Buy:** 평균적인 포지션의 10~30% 정도를 적용

**컨빅션 벳:** 손익비가 너무 좋고 셋업이 아름다운 경우, 평균적인 포지션의 2배 정도

평균적인 포지션을 10%로 놓고, 손익비 최소 1:2, 손절가 10%, 승률 50%라고 가정하면:

- 한 번 잃을 시 전체 포지션의 ~1%를 잃음 (1R)
- 한 번 벌면 전체 포지션의 2% 이상 (2R)

**1R = 한 트레이드에서 내가 감수하기로 한 고정된 손실 범위**

**하이베타에 집착하는 이유:** 회전율을 끌어올려 수익을 극대화하기 위해서

**초반에는 반복시행이 가능한 사이즈여야 합니다.** 벌고 나서 수익쿠션을 방패삼아 포지션 사이즈를 얼마간 키워보는 것은 괜찮습니다.

### 3. Kelly Criterion

**켈리 공식은 자산을 가장 빠르게 불리는 최적의 베팅 비율을 계산하는 공식입니다.**

f = (bp - q) / b

- f = 한번 베팅할 비율
- b = 손익비
- p = 승률
- q = 1-p

**이론적으로는 full kelly가 좋지만, 현실적으로는 위험합니다. 하방이 너무 열려있기 때문에 보통 half kelly 내지 1/4 kelly를 적용합니다.**

예시: 손익비 1:2(2R), 승률 50%라면

- Full kelly 적용 시 평균 포지션 사이즈: 25%
- Half kelly: 12.5%
