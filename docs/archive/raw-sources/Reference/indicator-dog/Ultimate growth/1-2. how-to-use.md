# 1-2. 🟢 Green (GO): "비중 실어!" (진입)
- 상태: [매수 최적기]

- 의미: 숲(GP)의 추세가 좋고, 나무(Sigma)의 타점도 완벽하게 일치하는 순간입니다.

- 행동: 적극적으로 매수하세요.

AI가 "지금이 가장 안전하고 수익 확률이 높은 구간입니다"라고 보증하는 상태입니다.

불타기(추가 매수)를 하거나 비중을 크게 실어도 좋은 자리입니다.

### 🟠 Orange (Ready): "준비해!" (관망)
- 상태: [폭풍전야]

- 의미: 당장 사라는 신호는 아니지만, 에너지를 꾹꾹 눌러 담고 있는 응축(Squeeze) 단계입니다.

- 행동: 지갑을 열 준비만 하세요.

스프링을 누르면 튀어 오르듯, 곧 위든 아래든 큰 방향성이 나옵니다.

절대 미리 사지 마세요! 초록불(Green)로 바뀌며 방향이 위로 잡히는 걸 확인하고 들어가야 안전합니다.

### 🔴 Red (Wait): "쉬어!" (휴식)
- 상태: [진입 금지]

- 의미: 하락 추세이거나, 뚜렷한 상승 모멘텀이 없는 지루한 구간입니다.

- 행동: 아무것도 하지 마세요.

> "쉬는 것도 투자다"라는 격언이 딱 맞는 구간입니다.

억지로 들어갔다가는 기회비용만 날리기 쉽습니다. 현금을 들고 다음 기회를 기다리세요.
