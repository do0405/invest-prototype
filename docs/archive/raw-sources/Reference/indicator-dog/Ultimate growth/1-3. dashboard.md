# 1-3. 대시보드 완전 정복: 차트의 내비게이션 📊
신호등이 '결과'라면, 대시보드는 그 결과가 나오게 된 '성적표'입니다.

차트 우측 하단에 있는 네모난 박스, 숫자가 많아서 당황스러우셨나요? 딱 두 가지만 체크하면 됩니다.

## 1. 성장 지표 (Growth): 회사가 돈을 잘 벌고 있나요?
주식의 기본은 결국 '실적'입니다. 아무리 차트가 좋아도 회사가 적자라면 모래 위에 지은 집과 같습니다. 대시보드 상단의 [성장] 테이블을 보세요.

EPS (순이익) & Sales (매출)

이 숫자들이 빨간색인가요? 🚨 경고! 회사의 성장이 멈췄거나 돈을 까먹고 있다는 뜻입니다. 차트가 좋아 보여도 비중을 줄이세요.

이 숫자들이 초록색인가요? ✅ 합격! 돈을 잘 벌고 있는 우량주입니다. 마음 편히 투자하세요.

## 2. 종합 점수 (Score): 지금 몇 점짜리 자리인가요?
AI가 이 종목의 추세, 수급, 실적, 패턴을 모두 분석해서 100점 만점으로 채점해 줍니다.

- 60점 이상: [안전권] 낙제점을 넘긴, 믿을만한 자리입니다. (Green 신호가 뜰 확률이 높습니다.)

- 30점 미만: [위험권] 아직은 때가 아닙니다. 점수가 오를 때까지 기다리세요.
